# Construction Issue Classifier

Use these categories for construction-industry issues:

- `cost`: materials, labor cost, inflation, project cost pressure
- `orders`: orders, bids, new contracts, public/private procurement
- `safety`: accidents, serious disaster law, safety compliance
- `tech`: smart construction, modular, BIM, AI, automation
- `labor`: workforce shortage, foreign workers, aging workforce
- `regulation`: government policy, laws, permits, public rules
- `market`: real estate, housing, market conditions, demand

Return only allowed labels. Prefer one or two labels unless the issue clearly crosses categories.
